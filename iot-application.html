SIT210 Developing IOT 2023

Activity 1 IoT applications:

a. What is an IoT application?

An IoT application or Internet of Things are applications which are used to control the hardware through the use of a simple interface build using web technologies such as python, html, css, js etc. Through this application the user is able to analyse and monitor sensor data.

b. What are the main Features needed?

1. Data Security 2. Real time data monitoring 3. Scalability 4. Integration 5. Optimization (Responsiveness) 6. Data backup 7. User Interface 8. Error reports notifications

c. What are some Examples?

1. Smart home security system 2. Smart evacuation system 3. Health monitoring system 4. Hospital or Restaurant Reservation system (Congestion avoiding system) 5. Appliance monitor and controller

Activity 2 Smart Building

a. What does smart mean?

Smart building refers to a building which is automated to a certain extent in order to reduce human effort and control appliances using an interface while also covering faults that may occur during the whole process. It also covers resource conservation, security and health of the users.

b. What are the main Features needed?

1. Data handling: data security, data backup, data access restriction 2. Sensor data precision and accuracy 3. Realtime health monitoring 4. Fault notifying system 5. Health preserving 6. Connectivity 7. Abstraction 8. Automation

c. What are some Smart-building scenarios

1. Security (Intruder detection) 2. Congestion avoidance detection 3. Air quality monitoring and ventilation 4. Turning on and off appliances automatically based on its usage hence saving electricity. 5. Contactless appliances 6. Smart Evacuation systems

Activity 3 Control Interface

a. What are the high-level features?

1. Automated lighting system 2. Security and access control 3. Smart maintenance system 4. Communication ease and connectivity 5. Resource usage control 6. Evacuation system 7. Reservation system

1. Sensors: Thermal sensors, ionization detectors, photoelectric detectors, motion sensors, pulse sensors 2. Camera 3. Signal transmitters 4. IoT compatible microcontrollers and microprocessors 5. Suitable storage (local or cloud) 6. Transmitters and receivers (M2M communication)

Who are the users?

1. Fire departments 2. Organizations (staff, employees, students) 3. Personal users 4. Logistical departments 5. Restaurants and hotels. 6. Technicians and security departments
